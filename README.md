## ローカル開発

1. リポジトリをクローン:
```bash
git clone https://github.com/i-repo-community/i-repo-direct-gateway.git
cd i-repo-direct-gateway
```

2. 依存関係をインストール:
```bash
npm install
```

3. サーバーを起動:
```bash
npm start
```
